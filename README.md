# ESP32-PN532

PN532 (HSU) library
- Platform: ESP-IDF (e.g. ESP32)
- Can work with DESFireAES library for NXP MIFARE DESFire EV1 cards
- See include file for details of functions

Includes KiCad PCB design for PN532 based board
